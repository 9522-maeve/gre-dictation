# GRE 随机默写

一个单文件 GRE 单词默写网页，打开即用，数据存本地浏览器。

## 使用方式

1. 下载 `index.html`，浏览器打开
2. 上传词书 `.txt` 文件（格式：每行 `英文 中文`，空格/Tab/逗号分隔均可）
3. 开始背单词

或者直接访问 GitHub Pages（部署后）。

## 功能

- **英→中（四选一）**：看英文选中文，支持键盘 1-4 快捷作答
- **中→英（拼写）**：看中文拼写英文，支持含连字符、分号分隔的多义项匹配
- **不重复抽词**：已背单词不再出现，全部背完提示后重新开始
- **自定义每轮词数**：10/20/30/50 词可选
- **错题本**：答错的词自动归入错题本，可查看、移除、复习
- **错题复习**：复习全部错题，不限词数，可反复练习
- **背诵进度追踪**：显示已背/总词数
- **跳过不计错题**：跳过不归入错题本，仅答错才记录
- **进度持久化**：词书和背诵进度自动保存在浏览器 localStorage

## 词书格式

```
abandon 放弃
abate 减弱
abbreviate 缩写
```

分隔符支持空格、Tab、逗号，多义项用分号或逗号分隔均可匹配。
